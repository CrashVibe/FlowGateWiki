---
title: 服务器绑定聊天平台
description: 将 Minecraft 服务器绑定到聊天平台，实现消息互通和指令交互
---

import { Step, Steps } from "fumadocs-ui/components/steps";

在经历前面的部署步骤后，我们已经成功搭建了 FlowGate Nexus 服务器和 Client 插件以及适配器。接下来，我们将介绍如何将 Minecraft 服务器绑定到一个聊天平台。

<Steps>
  <Step>
    ### 进入 FlowGate Nexus 服务器管理界面

    打开浏览器，访问你的 FlowGate Nexus 实例地址，进入管理界面。

  </Step>
  <Step>
    ### 点击你要绑定的服务器实例

    在服务器列表中，找到你需要绑定的服务器实例，点击对应卡片进入服务器配置页面。

  </Step>
  <Step>
    ### 进入基础设置

    在服务器配置页面，点击左侧菜单的 **基础设置** 选项，进入基础设置界面。

  </Step>
  <Step>
    ### 选择 Bot 实例

    在基础设置界面，找到 **Bot 实例** 配置项，点击下拉框选择你已经配置好的 Bot 实例。

  </Step>
  <Step>
    ### 保存配置

    选择完成后，点击页面底部的 **保存设置** 按钮，保存你的配置。

  </Step>
</Steps>

大功告成！现在你的 Minecraft 服务器已经成功绑定到聊天平台了。后续你设置的任何需要 Bot 实例的功能都可以在这个服务器实例上使用了，比如消息互通、指令交互等。
