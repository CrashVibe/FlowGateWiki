---
title: 配置目标
description: 添加和管理消息目标，后续功能可针对各目标进行单独配置
---

import { Step, Steps } from "fumadocs-ui/components/steps";

目标是 FlowGate 中的核心概念，代表一个聊天平台会话（如某个 QQ 群或私聊）。在此处添加目标后，消息互通、指令交互等功能均可针对每个目标进行单独配置。

<Steps>
  <Step>
    ### 进入目标配置页面

    在服务器配置页面中，点击左侧菜单的 **目标配置** 选项。

  </Step>
  <Step>
    ### 添加目标

    点击右上角的 **添加目标** 按钮，表格中会新增一行。依次填写以下信息：

    | 字段 | 说明 |
    | ---- | ---- |
    | **目标 ID** | 聊天平台的会话唯一标识，如 QQ 群号 |
    | **类型** | 选择 **群聊** 或 **私聊** |
    | **状态** | 选择 **已启用** 或 **已禁用** |

  </Step>
  <Step>
    ### 保存配置

    确认信息无误后，点击页面底部的 **保存配置** 按钮。

    > 点击 **取消更改** 可放弃本次编辑，恢复上次保存的状态。

  </Step>
</Steps>

## 管理目标

- **删除目标**：点击对应行右侧的 **删除** 按钮，然后保存配置
- **临时禁用**：在"状态"列选择 **已禁用**，可保留配置但暂停该目标的任何功能
