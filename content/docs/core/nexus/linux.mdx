本指南将带你在 **Linux** 系统上完成 FlowGate 的完整部署，包括 **FGateNexus**（核心服务）和 **FGateClient**（Client 端）两个部分。

## 前置要求

- Linux（x64 或 ARM64）操作系统
- 基本的命令行操作能力

<Steps>
  <Step>
    ## 下载 FGateNexus

    前往 [FGateNexus Releases](https://github.com/CrashVibe/FGateNexus/releases/latest) 下载适用于 Linux 的压缩包，根据你的服务器架构选择对应版本：

    | 架构  | 适用场景              | 文件名                                                                              |
    | ----- | --------------------- | ----------------------------------------------------------------------------------- |
    | x64   | 大多数 VPS / 物理机   | <VersionFilename repo="nexus" template="FGateNexus-{version}-linux-x64.tar.gz" />   |
    | ARM64 | ARM 服务器 / 树莓派等 | <VersionFilename repo="nexus" template="FGateNexus-{version}-linux-arm64.tar.gz" /> |

    下载完成后，将压缩包解压到你希望运行服务的目录，例如 `/opt/FlowGate/`：

    ```bash
    mkdir -p /opt/FlowGate
    tar -xzf FGateNexus-*-linux-*.tar.gz -C /opt/FlowGate
    cd /opt/FlowGate
    ```

    解压后的目录结构大概如下：

    <Files>
      <File name="FGateNexus" />
      <Folder name="config" defaultOpen>
        <File name="appsettings.json" />
      </Folder>
    </Files>

  </Step>
  <Step>
    ## 配置 FGateNexus

    打开 `config/appsettings.json`，根据你的需求修改配置项。

    <Callout type="info">首次运行时，FGateNexus 会自动在目录下创建 `sqlite.db` 数据库文件以及配置文件。</Callout>

  </Step>
  <Step>
    ## 运行 FGateNexus

    赋予可执行权限后运行：

    ```bash
    chmod +x /opt/FlowGate/FGateNexus
    cd /opt/FlowGate
    ./FGateNexus
    ```

    启动成功后，你将看到类似以下的输出：

    ```log
    [MIGRATION] 检查并执行数据库迁移...
    [SUCCESS] 数据库准备就绪
    INFO: 配置文件路径：.../config/appsettings.json
    INFO: [REGISTER] 请求处理器已注册：player.login
    Listening on http://127.0.0.1:3000/...
    INFO: Koishi 服务启动 http://127.0.0.1:5140
    ```

    服务启动后会监听以下端口：

    | 服务                  | 地址                    |
    | --------------------- | ----------------------- |
    | FGateNexus HTTP / API | `http://127.0.0.1:3000` |
    | Koishi                | `http://127.0.0.1:5140` |

    <Callout type="tip">Koishi 用于与你的聊天适配器进行链接，在管理面板中配置适配器时需要使用到这个地址。</Callout>

  </Step>
  <Step>
    ## 安装 FGateClient 插件

    插件安装步骤与操作系统无关，请参阅独立教程：[安装 FGateClient 插件](/docs/core/client/plugin)。

  </Step>
</Steps>

## 开机自启（可选）

推荐使用 systemd 将 FGateNexus 注册为系统服务。

新建文件 `/etc/systemd/system/fgatenexus.service`，内容如下：

```ini
[Unit]
Description=FGateNexus Service
After=network.target

[Service]
Type=simple
WorkingDirectory=/opt/FlowGate
ExecStart=/opt/FlowGate/FGateNexus
Restart=on-failure
RestartSec=5

[Install]
WantedBy=multi-user.target
```

然后启用并启动服务：

```bash
systemctl daemon-reload
systemctl enable fgatenexus
systemctl start fgatenexus
```

查看运行状态：

```bash
systemctl status fgatenexus
```
